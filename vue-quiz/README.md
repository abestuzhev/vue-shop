## Общее описание проекта

Разработаны две версии простого опросника для факторинговых компаний, демонстрирующие различные подходы к работе с Vue 3:
- Composition API - современный подход с использованием функций ref, computed, reactive
- Options API - классический подход, совместимый с Vue 2

Обе версии реализуют одинаковый функционал:
- Загрузка данных опросника с внешнего API (https://bc8aa7f27ab0c7c8.mokky.dev/quiz)
- Навигация по вопросам с кнопками "Назад" и "Далее"
- Два типа вопросов: Boolean (да/нет) - отображаются кнопками, Number - отображаются полем ввода Валидация ответов - кнопка "Далее" активна только при выбранном ответе
- Отображение результатов - сводка всех ответов
- Форматирование чисел - функция numberWithSpace для разделения тысяч пробелами
- Конвертация ответов - функция convertAnswer для отображения ответов в читаемом виде


### Composition API (quiz-composition-api.js)
Использует функцию setup() как точку входа Реактивные данные объявлены через ref() и reactive() Вычисляемые свойства через computed() Хуки жизненного цикла через onMounted() Более функциональный стиль с явным возвратом всех свойств Использует reactive() для вопросов для глубокой реактивности

### Options API (quiz-option-api.js)
Классическая структура с секциями data, computed, methods, created Данные объявлены в data() как обычные свойства Вычисляемые свойства в секции computed Методы в секции methods Хуки жизненного цикла как отдельные методы Более императивный стиль с использованием this
