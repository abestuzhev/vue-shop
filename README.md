## Vue shop

### Цели и задачи проекта
- Изучение работы Vue 3 - понимание реактивной системы и жизненного цикла компонентов
- Передача данных между компонентами - освоение props, emits, provide/inject
- Работа с API - выполнение запросов на бэкенд и обработка JSON-данных
- Маршрутизация - реализация переходов между страницами с помощью Vue Router
- Работа с глобальным стором Pinia, на примере светлой/темной темы сайта
- Стилизация - применение TailwindCSS для создания современного UI
- Анимации - создание красивых и простых анимаций для списков товаров, корзины и других элементов

### Стэк технологий:
- Vue 3
- Pinia
- Vue Router
- TailwindCSS
- Axios
- @formkit/auto-animate
- Composition API / Options API
- ESLint
- Prettier
- Vite
